# carsmgr
demo firebase web + node js

Require:
1. install express:
$ npm install express --save
2. install deamon:
$ npm install nodemon --save-dev

You can run 
$ npm run dev 
to trigger nodemon server.js.
3. install mongodb:
$ npm install mongodb --save
4. install Embedded JavaScript:
$ npm install ejs --save
